About CAS

The Center for Archaeology and Society (CAS) at Arizona State University supports collections care, education, public outreach, and research within the School of Human Evolution and Social Change.

This homepage provides a public directory of software, services, and infrastructure repositories maintained by the CAS GitHub organization.

Visit the CAS website View the GitHub organization

Featured Project

ArchaeoDash

ArchaeoDash is an analytical dashboard for grouping and assessing neutron activation analytical chemical compositional data in archaeological research workflows.

Repository

Secondary Feature

heuristR

heuristR is an R client for Heurist databases with session-based authentication, metadata helpers, safe write workflows, and rollback support for reversible scripted changes.
